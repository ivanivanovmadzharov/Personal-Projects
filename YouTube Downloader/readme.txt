YouTube Downloader 

---------------------------------------------------------------------------

NOTE: You should respect YouTube's terms of service and copyright rules when downloading videos. This code is provided for educational purposes, and you should not use it to download copyrighted content without permission.

---------------------------------------------------------------------------

How to download video from YouTube:
1. Open the Command Prompt
2. Type "cd " and then the directory of the python file and click Enter
3. Type "python youtube_downloader.py" and click Enter
4. Type the URL of the wanted video and click Enter
5. If the URL is correct, the video will be downloaded in the same directory of the python file
